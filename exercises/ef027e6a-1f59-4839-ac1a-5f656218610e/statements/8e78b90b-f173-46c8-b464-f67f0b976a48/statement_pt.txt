<p>As consultas multitarefa podem incluir tantas composições de tabela conforme necessário para alcançar o resultado desejado. Se os dados envolvidos na solução de uma consulta pertencem a várias tabelas, você terá que seguir o caminho necessário para relacionar todos os dados envolvidos na consulta.</p><h3>Exemplo:</h3><p>Se for pedido uma consulta do tipo Qual é o nome dos departamentos localizados na região cujo nome é 'Europa'?, os atributos que intervêm nessa declaração são: <code>DEPARTMENT_NAME</code> e <code>REGION_NAME</code>.</p><p>De acordo com a imagem a seguir, para obter de um atributo para o outro, devemos usar as seguintes tabelas: <code>REGIONS</code>, <code>COUNTRIES</code>, <code>LOCATIONS</code> e <code>DEPARTMENTS</code></p><p><img alt="HR schema" src="https://www.oracle.com/webfolder/technetwork/tutorials/obe/db/12c/r1/odb_quickstart/images/t10101.gif" style="height:426px; width:600px"/></p><table align="center" border="1" summary="Results from query 1"><tbody></tbody></table><h3>Exercício</h3><p>Mostra os departamentos que estão localizados na região cujo nome é <code>'Europe'</code>.</p><p>Encomende os resultados alfabeticamente pelo nome do departamento.</p>

<p>Para manipular dados existentes em um <strong>banco de dados relacional</strong>, vamos usar a linguagem <strong><big>SQL</big></strong>. Em particular, se o que queremos é extrair um subconjunto dos dados existentes, usaremos o comando <code>SELECT</code>.</p><p>A primeira coisa que vamos fazer é mostrar todos os dados em uma determinada tabela, sem restrições.</p><h3>Exemplo:</h3><p>se quisermos mostrar todos os dados na tabela <code>regions</code> Vamos fazê-lo.</p><blockquote><p><code>SELECT * FROM regions;</code></p></blockquote><p>Tudo o que precisamos fazer é escrever o nome da tabela (<code>regions</code>) por trás do texto SQL <code>SELECT * FROM</code>, obtendo a seguinte tabela:</p><table border="1px solid blue"><tbody><tr><th>REGION_ID</th><th>REGION_NAME</th></tr><tr><td>1</td><td>Europa</td></tr><tr><td>2</td><td>América do Sul</td></tr><tr><td>3</td><td>Ásia</td></tr><tr><td>4</td><td>Oriente Médio e África</td></tr></tbody></table><h3>Exercício:</h3><p>De acordo com o exemplo acima, ele mostra todos os dados na tabela <code>countries</code>.

<p>När vi fastställer det attribut genom vilket vi kommer att beställa resultaten, kan vi ange att ordern stiger upp (<code>ASC</code> eller ingenting) eller ättling (<code>DESC</code>) )</p><blockquote><p><code>SELECT <em>atributos</em><br/>FROM <em>tabla</em><br/>ORDER BY <em>atributo </em>[DESC | ASC ];</code></p></blockquote><h3>Exempel:</h3><p>Vi kan verifiera att lägga till ASC till den tidigare meningen som beställde regionerna genom deras REGION_NAME, inte varierar resultatet.</p><blockquote><p><code>SELECT * FROM<br/>regions<br/>ORDER BY REGION_NAME ASC;</code></p></blockquote><table align="center" border="1px solid" summary="Results from query"><tbody><tr><th>REGION_ID</th><th>REGION_NAME</th></tr><tr><td>2 2 2 2 2</td><td>Amerika</td></tr><tr><td>3 3 3 3 3</td><td>Asien Asien</td></tr><tr><td>1 1</td><td>Europa</td></tr><tr><td>4.4 4.</td><td>Mellanöstern och Afrika</td></tr></tbody></table><p>Å andra sidan, genom att lägga till DESK lyckades vi vända ordern som visar resultatet:</p><blockquote><p><code>SELECT * FROM<br/>regions<br/>ORDER BY REGION_NAME DESC;</code></p></blockquote><table align="center" border="1px solid" summary="Results from query"><tbody><tr><th>REGION_ID</th><th>REGION_NAME</th></tr><tr><td>4.4 4.</td><td>Mellanöstern och Afrika</td></tr><tr><td>1 1</td><td>Europa</td></tr><tr><td>3 3 3 3 3</td><td>Asien Asien</td></tr><tr><td>2 2 2 2 2</td><td>Amerika</td></tr></tbody></table><h3>Övning:</h3><p>Visar namn (DEPARTMENT_NAME) på avdelningarna (avgångar) sorterade <strong>nedåt </strong>vid namn.</p>
